异同

相同

  • 参数入口都是父类句柄
  • 实际执行都是子类方法

不同

  • Upcasting 父类和子类都有该方法(即使用的是重写的方法)
  • Downcasting 父类不具备该方法,即子类使用的是特有方法.

因此,如果使用的是子类特有方法,那么采用DownCasting;如果使用的子类重写的方法,则采用upCasting.
